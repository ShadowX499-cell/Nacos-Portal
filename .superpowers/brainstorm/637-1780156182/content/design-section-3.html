Design — Section 3 of 4: Frontend Pages & Components

New pages and component changes across the admin and student portals.

Admin Portal — New & Changed Pages

CHANGED — CreateUserPage

Add passport photo uploader (drag-and-drop + preview), Date of Birth picker, State dropdown (36 Nigerian states), LGA dropdown (filtered by state), Home Address textarea. All optional except photo — admin can skip and student updates later.

CHANGED — UserListPage

Add Program tab row (All · ICT · CSC · CRE) beside the existing Level filter. Student rows show passport thumbnail. StudentStatus badge (Active / Carryover / Graduated / Suspended) shown beside name.

CHANGED — AdminDashboard

"Students by Level" donut gains a sibling: "Students by Program" donut (ICT/CSC/CRE). Total Revenue KPI card becomes a clickable link → /admin/revenue. Dashboard shows current session/semester prominently with an "Advance" button for HOD.

NEW — RevenuePage /admin/revenue

Filter bar (date range, session, semester, program, level, type). KPI row: Total · NACOS Dues · School Fees · Result Subscriptions · Collection Rate %. Bar chart by month. Breakdown donuts by type & program. Paginated payment table. CSV export button.

NEW — AuditLogPage /admin/audit-logs

HOD only. Filter bar: actor name, role, action type, entity type, date range. Timeline table with actor avatar, action badge, entity link, timestamp. Click any row → expand before/after JSON diff. CSV export.

NEW — SuperAdminsPage /admin/super-admins

HOD only. List of all super admins with role badge. "Add Super Admin" button — modal to pick an existing admin and assign role. Revoke button with confirmation. Role capabilities summary shown inline.

NEW — AcademicCalendarPage /admin/academic

HOD only. Shows current session + semester. "Preview Advance" button → modal with dry-run numbers (X active, Y will graduate, Z carry-overs). Confirm button triggers the transition. History table of all past advances.

Student Portal — Changed Components

CHANGED — StudentDashboard hero

Replace initials avatar with passport photo (falls back to initials if none). The 3 stat cards become 6: Credits Earned · Semesters · Results · NACOS Due (X/8) · School Fees (X/8) · Course Form (X/8). Carryover badge shown on profile line if applicable.

CHANGED — StudentProfilePage

Student can view and update their own passport photo, state, LGA, DOB, home address. Read-only fields: name, student ID, email, program, level, student status.

NEW — CarryoverSection on ResultsHomePage

If student has carry-over courses, a prominent banner lists them: course code, title, original session, status (Pending / Resolved). Links to the relevant gradebook result.
